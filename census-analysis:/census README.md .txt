# Census Data Analysis

## Overview
This project explores population and socio-economic patterns using census data. The analysis focuses on understanding demographic distributions and identifying trends through exploratory data analysis and visualization.

## Problem Statement
The aim of this project is to analyze census data to uncover meaningful insights related to population structure, employment, education, and regional variation.

## Dataset
- Source: University-provided census dataset
- Description: Demographic and socio-economic attributes including age, employment status, education level, and geographic region
- Note: Dataset not uploaded due to usage restrictions

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Methodology
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visual analysis of demographic and socio-economic variables
- Interpretation of patterns and trends

## Key Findings / Results
- Identified significant demographic variations across regions
- Observed patterns linking education level and employment status
- Highlighted population distribution trends across age groups

## Visualisations
Key plots and charts are included in the notebook and stored as images for reference.

## How to Run
1. Clone the repository
2. Open the Jupyter Notebook
3. Run all cells sequentially
